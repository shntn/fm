# Code Style

- Rust : OS, Liabrary などの下層と Lua スクリプトで実装するアプリケーションをつなぐインタフェース層
- Lua : ファイルマネージャとしての制御ロジックを実装。必要な情報はインタフェース層が提供する API を使用
- 関数は一つのことだけを行うべきです。「そして」という言葉を使って説明する必要がある場合は、関数を分割してください
- 変数にはその内容に基づいて名前を付け、関数にはその動作に基づいて名前を付ける。
- 名前を省略しないでください。`getUserProfile` を `getUsrProf` のように省略しないでください。簡潔さよりも明瞭さが重要です
- 不要なコードはコメントアプトではなく、削除してください。Gitが記憶しています
- エラーは明示的に処理してください。例外を無視したり、エラーの戻り値を無視したりしないでください
